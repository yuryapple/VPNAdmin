# VPNAdmin

1. Create DB on MySQL Server from vpnDB.sql 

2. Change this section of config.txt  
    
    [Data Base]<br>
    dataBase = vpnDB	            ;  Name of dataBase <br>
    host = localhost<br>
    user = root<br>
    password =	MyPass				;  password<br>

3. On browser enter name of site. (In my case localhost).
